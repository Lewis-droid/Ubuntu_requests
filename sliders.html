PROJECT OVERVIEW

The Cyberpunk 2077-themed portfolio website is designed to showcase Lennox Lewis's skills and projects with a futuristic, eye-catching design.

  • Responsive design that works on mobile and desktop
  • Cyberpunk aesthetic with neon colors and glitch effects
  • Five main sections with smooth navigation
  • Interactive elements with hover effects
  • Modern CSS animations and transitions

The website serves as both a portfolio and a demonstration of front-end development skills with attention to UI/UX principles.

TECHNICAL IMPLEMENTATION

Key Features

Responsive Design

Media queries adapt layout for mobile, tablet, and desktop screens

Smooth Navigation

JavaScript-enabled scrolling to sections

CSS Animations

Keyframe animations for glitch and shine effects

Interactive Elements

Hover states and transitions for engagement

Code Structure

/* CSS Variables for consistent theming */
:root {
  --neon-pink: #ff2d95;
  --neon-blue: #00f3ff;
  --dark-bg: #0a0a16;
}

/* Glitch animation keyframes */
@keyframes glitch {
  0% { text-shadow: 0 0 20px var(--neon-blue); }

  10% { text-shadow: 0 0 20px var(--neon-blue); }

}

WEBSITE SECTIONS

The portfolio is organized into five main sections, each with a specific purpose.

  • Header: Fixed navigation with smooth scrolling links
  • Hero Section: Introduction with call-to-action button
  • About Section: Bio with contact information and holographic visual
  • Projects Section: Grid layout of project cards with hover effects
  • Contact Section: Form and social media links
  • Footer: Copyright and additional navigation

Mobile Responsiveness

The website features a mobile-friendly hamburger menu that transforms the navigation on smaller screens, ensuring optimal user experience across all devices.

Animation Examples

// Smooth scrolling function
document.querySelectorAll('a[href^="#"]').forEach(anchor => {
  anchor.addEventListener('click', function(e) {
    e.preventDefault();
    const target = document.querySelector(this.getAttribute('href'));
    window.scrollTo({
      top: target.offsetTop - 80,
      behavior: 'smooth'
    });
  });
});
